Практическое задание по теме “NoSQL”

1. В базе данных Redis подберите коллекцию для подсчета посещений с определенных IP-адресов.

SADD guest_ip '127.0.0.1' '127.0.0.2' '127.0.0.3' '127.0.0.3'

Задаем множество IP адресов, с которых быдем считать посещения

SISMEMBER '127.0.0.1'

Проверем, входит IP адрес во множество. Если входит, то храним список счетчик посещений (или дату/время посещения) с адреса и адрес в упорядоченном множестве: 

ZADD guests_ip 00001 '127.0.0.1'
